## Phase Goal (MVP user story)

**As a** desktop user, **I want to** switch to any theme (static preset or matugen dynamic) and have every visible app re-theme live from one shared engine with the stuck-white bug gone, **so that** my whole desktop stays consistent without relogins and theming fixes stop looping.

> After this plan, a real user gains: a single command/keybind that actually switches the visible desktop (Hyprland, waybar, kitty, swaync) to any of the 7 themes live — and it works identically whether they pick it, pick a wallpaper, or log in — with git staying clean afterward.
